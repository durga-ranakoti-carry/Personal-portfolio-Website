# Personal Portfolio Website 🌐

A fully responsive personal portfolio website built using HTML, CSS and JavaScript.

## 🔗 Live Preview
Open index.html in your browser to view the project.

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Boxicons

## ✨ Features
- ✅ Animated Navbar with smooth transitions
- ✅ Hero Section with typing animation
- ✅ About Me Section
- ✅ Education & Projects Timeline
- ✅ Skills Section with progress bars
- ✅ Contact Form
- ✅ Scroll animations on every section
- ✅ Fully Responsive Design
- ✅ Sticky Header on scroll

## 📁 Project Structure
- index.html — Main HTML file
- style.css — All CSS styling
- script.js — JavaScript functionality

## 🚀 How to Run
1. Download all files
2. Keep all files in same folder
3. Open index.html in browser
